Projects

RDFMapper Public

This project implements a flexible RDF conversion tool using the Strategy Design Pattern in Python. It supports parsing data from multiple input formats (JSON, XML, SQL) and converting them into RDF or OWL-compatible formats using rdflib. A lightweight HTML GUI is provided using Flask, making it accessible from a web browser.

Lifesizev3 Public

This is a PowerShell GUI Tool developed in 2013 to improve video connection management for OMP. The tool was created as an alternative to the browser-based application, which frequently crashed, making video connections unreliable. Over the course of three years, this tool was actively used by the team to ensure stable and seamless video connections until it was eventually retired. The tool operates via PowerShell, offering a more stable and efficient way to establish connections for video conferencing and streaming, addressing the performance issues with the browser app.

CSVtoSQLCreate Private

CSVtoSQLCreate is a Python-based web tool that allows users to upload or paste CSV data and automatically generates SQL CREATE TABLE and INSERT statements using metadata embedded in the CSV file itself. The application is built using Flask and does not rely on JavaScript frameworks, ensuring simplicity, transparency, and ease of use.

NetWhirl Private

NetWhirl is a lightweight Chrome extension that takes users on a digital adventure with a single click. When you hit the "Whirl" button, it instantly redirects you to a randomly curated website featuring fascinating, educational, or thought-provoking content. Designed to spark curiosity and break the routine, NetWhirl is perfect for anyone looking to discover something new with every spin.

RDFER Private

RDFER is a custom Outlook plugin that extracts email metadata and content, converts the information into RDF (Resource Description Framework) format, and sends it directly to a Jena Fuseki server for semantic processing and storage. Designed to integrate email communication into linked data environments, it enables automated knowledge graph enrichment and semantic search capabilities within enterprise ecosystems.

AutoBitTrading Private

An automated cryptocurrency trading algorithm written in Python that interfaces with the Binance API to execute real-time Bitcoin trades. The system analyzes market conditions, places buy/sell orders based on predefined strategies, and includes basic risk management logic for unattended trading. Ideal for exploring algo-trading concepts and building custom trading bots on Binance

VOIPwebapp Private

The VOIPwebapp is a web-based calculator designed to help businesses and IT professionals estimate the bandwidth and infrastructure requirements for deploying VoIP (Voice over IP) solutions. Users can input variables such as the number of concurrent calls, codec types, and overhead to receive detailed network usage projections. Ideal for capacity planning, system design, and demonstrating ROI on VoIP deployments.
